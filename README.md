# soloish-skills

A small collection of personal agent skills.

## Install

macOS / Linux:

```bash
curl -fsSL https://raw.githubusercontent.com/soloish90/soloish-skills/main/install.sh | bash
```

Windows PowerShell:

```powershell
irm https://raw.githubusercontent.com/soloish90/soloish-skills/main/install.ps1 | iex
```

## Skills

### session-handoff

Creates a concise Markdown handoff document from the current session context so a fresh agent session can continue the work without inherited chat history and context.
